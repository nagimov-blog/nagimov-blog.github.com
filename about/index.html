About


I try not to blog unless what I am trying to convey could be useful and is not googleable. It is usually about tech and engineering. It is not meant to solve everyone’s problems, just something I struggle with on a daily basis.

English is my third language, and this shows. Please let me know if you find any such issues.

My current contacts can be found at nagimov.com (javascript required).

Javascript and user-tracking

Pages on the site include the following javascript libraries:

  • jquery and bootstrap as a part of Hugo theme
  • remark commenting system (https://remark42.com), which doesn’t spy on users
  • open web analytics software (http://www.openwebanalytics.com), which shares none of its tracking data with anything else

Feel free to disable javascript entirely, the content will always be accessible as plain html.
